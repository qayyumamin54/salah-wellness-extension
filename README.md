# 🕌 Salah & Wellness Break

A Chrome extension that reminds you to pray **and** take care of your body — at the same time.

When prayer time arrives, a full-screen overlay appears on whatever website you're browsing, guiding you through a short wellness break (stretching, neck rolls, deep breathing) before you go perform your solat.

---

## ✨ Features

- 🕐 **Accurate prayer times** — powered by Aladhan.com based on your city
- 🧘 **Forced wellness break** — full-screen overlay with step-by-step guided stretches
- ⏱ **Timed steps** — each step has a countdown so you actually do it
- 🔔 **Two notification styles** — popup or full-screen overlay
- 🌐 **Bilingual** — English and Bahasa Melayu
- 💾 **100% local** — no accounts, no servers, no tracking

---

## 🚀 Install

Available on the [Chrome Web Store](#) *(link coming soon)*

Or load unpacked locally:
1. Download and unzip this repo
2. Go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked** → select the `salah-wellness` folder

---

## 🔒 Privacy

No data is collected or sent to any server. Your city coordinates are only used to fetch prayer times from the public Aladhan API. See the full [Privacy Policy](./privacy-policy.html).

## 🛠 Built With
